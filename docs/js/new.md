# new 绑定

> 函数如果作为构造函数使用 `new` 调用时， `this` 绑定的是新创建的构造函数的实例。

```js
function Foo() {
  console.log(this)
}

var bar = new Foo() // 输出: Foo 实例，this 就是 bar
```

> 实际上使用 new 调用构造函数时，会依次执行下面的操作：

- 创建一个新对象
- 构造函数的 prototype 被赋值给这个新对象的 **\_\_proto\_\_**
- 将新对象赋给当前的 this
- 执行构造函数
- 如果函数没有返回其他对象，那么 new 表达式中的函数调用会自动返回这个新对象，如果返回的不是对象将被忽略
